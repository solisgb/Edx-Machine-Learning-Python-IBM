K-Medias Clustering
¶

Introduction¶

Existen muchos modelos para clustering. En este lab, estaremos presentando el modelo que es considerado el ms simple de todos. Aunque simple, el K-medias se utiliza mucho para clustering en varias aplicaciones de ciencia de datos, especialmente si se necesita de forma rpida descubrir nuevas conclusiones a partir de datos no etiquetados. En este lab, aprender a utilizar k-Medias para segmentar clientes.

Algunas aplicaciones de k-medias del mundo real:

  • Segmentacion del cliente
  • Entender lo que los visitantes de un sitio web intentan realizar
  • Reconocimiento de patrones
  • Machine learning
  • Compresin de datos

En este lab practicamos clustering k-medias con 2 ejemplos:

  • k-means en un dataset generado al azar
  • Usando k-medias para la segmentacin del cliente

k-Media en un dataset generado aleatoriamente¶

Creemos nuestro propio dataseet para este lab!

Necesitamos primero configurar una semilla aleatoria(random seed). Utilizaremos la funcin numpy's random.seed(), donde la semilla se establecer con el valor 0

In [3]:
np.random.seed(0)

Luego, haremos clusters aleatorios de puntos usando la clase make_blobs . La clase make_blobs puede aceptar varias entradas, pero estaremos usando concretamente estas.

Entrada

  • n_samples: El nmero total de puntos equitativamente divididos entre los clusters.
    • El valor ser: 5000
  • centers: El nmero de centros a generar.
    • El valor ser: [[4, 4], [-2, -1], [2, -3],[1,1]]
  • cluster_std: El desvo estndar de los clusters.
    • El valor ser: 0.9

Salida
  • X: Arreglo de la forma [n_samples, n_features]. (Matrz de Distancia)
    • Muestras generadas.
  • y: Arreglo de la forma [n_samples]. (Response Vector)
    • Las etiquetas de nmeros enteros para la pertenencia de cluster en cada muestra.
In [4]:
X, y = make_blobs(n_samples=5000, centers=[[4,4], [-2, -1], [2, -3], [1, 1]], cluster_std=0.9)

Estableciendo K-Medias¶

Ahora que tenemos nuestros datos aleatorios, configuremos nuestro Clustering K-Medias.

La clase KMedias tiene muchos parmetros que se pueden utilizar, pero estaremos usando estos tres:

  • init: Mtodo de inicializacin de los centroides.
    • El valor ser: "k-means++"
    • k-means++: Elije centros de clusters iniciales eficientes para el clustering k-media de forma tal de acelerar la convergencia.
  • n_clusters: El nmero de clusters a formar y la cantidad de centroides a generar.
    • El valor ser: 4 (tenemos 4 centros)
  • n_init: Cantidad de veces que el algoritmo k-medias se ejecutar con diferentes semillas centroides. El resultado final ser la mejor salida de consecutivas ejecuciones de n_init en trminos de inercia.
    • Value will be: 12

Inicializar KMedias con estos parmetros, donde el parmetro de salida se llama k_means.

In [6]:
k_means = KMeans(init = "k-means++", n_clusters = 4, n_init = 12)

Ahora, unamos el modelo KMedias con la matriz de distancia que creamos anteriormente, X

Creando la Trama¶

En este momento, tenemos los datos generados al azar y el modelo KMedias inicializado. Podemos dibujarlos y ver de qu se trata!

Favor leer todo el cdigo y los comentarios para entender cmo dibujar el modelo.

In [11]:
# Inicializar el dibujo con las dimensiones especificadas.
fig = plt.figure(figsize=(6, 4))

# Los colores usan un mapa de color, dnde produciremos un arreglo de colores basados en
# el nmero de etiquetas que hay. Usaremos set(k_means_labels) para obtener
# etiquetas unvocas.
colors = plt.cm.Spectral(np.linspace(0, 1, len(set(k_means_labels))))

# Crear un dibujo
ax = fig.add_subplot(1, 1, 1)

# Loop For que dibuja los puntos de datos y los centroides.
# k tomar valores entre 0-3, los cuales coincidirn con los clusters posibles en el
# que est cada punto.
for k, col in zip(range(len([[4,4], [-2, -1], [2, -3], [1, 1]])), colors):

    # Crear una lista de todos los puntos, donde aquellos que estn 
    # en el cluster (ej. cluster 0) estn etiquetados como verdadero, o en su defecto
    # estarn etiquetados como falso.
    my_members = (k_means_labels == k)
    
    # Definir el centroide o centro del cluster.
    cluster_center = k_means_cluster_centers[k]
    
    # Dibjuar los puntos de datos con color col.
    ax.plot(X[my_members, 0], X[my_members, 1], 'w', markerfacecolor=col, marker='.')
    
    # Dibujo de los centroides con un color especfico pero una linea ms oscura
    ax.plot(cluster_center[0], cluster_center[1], 'o', markerfacecolor=col,  markeredgecolor='k', markersize=6)

# Ttulo del dibujo
ax.set_title('KMeans')

# Eliminar los ticks del eje x
ax.set_xticks(())

# Eliminar los ticks del eje y
ax.set_yticks(())

# Mostrar el dibujo
plt.show()
