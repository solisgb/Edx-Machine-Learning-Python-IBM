Density-Based Clustering
¶

La mayora de las tcnicas tradicionales de clustering, tales como k-Means,jerrquicas, y el fuzzy clustering, se pueden utilizar para agrupar datos de una forma no supervisada.

Sin embargo, cuando se aplica a tareas con clusteres de forma arbitraria,o clusteres dentro de clusteres, las tcnicas tradicionales podran no ser capaces de lograr buenos resultados. Es decir, los elementos en el mismo cluster pueden no compartir suficiente similitud -- o el rendimiento puede ser pobre. Adems, el Clustering basado en densidad ubica regiones de alta densidad que se separan una de otra por regiones de baja densidad. Densidad, en este contexto, se define como el nmero de puntos dentro de un radio especfico.

En esta seccin, el principal foco ser manipular los datos y las propiedades del DBSCAN y observar el clustering resultante.

Generación de datos¶

La siguiente funcin genera los puntos de datos y necesita estas entradas:

  • centroidLocation: Coordina los centroides que generarn los datos aleatorios.
    • Example: input: [[4,3], [2,-1], [-1,4]]
  • numSamples: El nmero de datos que queremos generar, dividir sobre los centroides (# de centroides definidos en centroidLocation)
    • Ejemplo: 1500
  • clusterDeviation: La desviacin standard entre clusters. Mientras ms grande el nmero, ms lejos la distancia.
    • Ejemplo: 0.5
In [6]:
def createDataPoints(centroidLocation, numSamples, clusterDeviation):
    # Crear datos aleatorios y almacenar en la matriz X y el vector y.
    X, y = make_blobs(n_samples=numSamples, centers=centroidLocation, 
                                cluster_std=clusterDeviation)
    
    # Estandarizar caractersticas eliminando el promedio y la unidad de escala
    X = StandardScaler().fit_transform(X)
    return X, y

Modelado¶

DBSCAN significa Density-based Spatial Clustering of Applications with Noise (Clustering espacial basado en densidad de aplicaciones con ruido). Esta tcnica es una de las ms comunes de algoritmos de clustering, que funciona en base a la densidad del objeto. DBSCAN trabaja en la idea de que si un punto en particular pertenece a un cluster, debera estar cerca de un montn de otros puntos en ese cluster.

Funciona en base a 2 parmetros: Radius (Radio) y Puntos Mnimos.
Epsilon determina un radio especificado que, si incluye suficientes puntos dentro de l, lo llamamos de "rea densa". minimumSamples determina el nmero mnimo de puntos de datos que queremos en un neighborhood (vecindario) para definir un cluster.
